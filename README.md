# 한땀한땀

## Project Description

- 해외 한식당 입점 증가 추세이지만 한식당 정보가 부정확하다.
- 사용자 소재 지역의 한식당을 추천, 정보를 제공한다.


## Built with

- Java
- Spring MVC
- Gradle
- MyBatis
- MySQL
- JSP
- Ajax
- JQuery

## Usage

- 국적에 따른 가입으로 같은 국적 이용자의 리뷰를 볼 수 있다.
- 식당 리뷰를 통해 별점을 메길 수 있다.
- 즐겨찾기를 등록하여 자주 찾는 식당을 모아볼 수 있다.
- 예약 기능을 통해 고객은 예약을, 업주는 예약 승인/거절을 할수 있다.
